# Lidermed Platform — корпоративный сайт

Корпоративный веб-сайт компании — разработчика IT-решений для фармацевтической отрасли (по ТЗ в `site.md`).

## Структура

- **Публичная часть:** главная, о компании, услуги, отрасль (фарма/GxP), кейсы, технологии, контакты, блог, политика конфиденциальности.
- **Формы:** обратная связь на странице «Контакты» (консультация, КП), согласие на обработку персональных данных.
- **Дополнительно:** уведомление о cookie, поиск по сайту, адаптивная вёрстка (mobile-first).
- **Админ-панель:** вход по логину/паролю; просмотр заявок и **экспорт в CSV**; **управление страницами** (контент, meta, порядок в меню); **контент главной** (hero, преимущества, цифры); **кейсы и новости** (добавление, редактирование, публикация/снятие); **настройки** (контакты, соцсети, мета, SEO, email для уведомлений о заявках, SMTP); **пользователи и роли** (редактор/администратор). По умолчанию: логин **admin**, пароль **admin123**.

## Запуск

### Только статика (без сервера)

Откройте в браузере `index.html` или разместите папку на любом веб-сервере. Формы покажут сообщение об успешной отправке (без сохранения на сервер).

### С сервером (заявки + админка)

```bash
npm install
npm start
```

- Сайт: http://localhost:3000  
- Админ-панель: http://localhost:3000/admin  

Заявки сохраняются в `data/leads.json`, настройки — в `data/settings.json`.

## Файлы

| Файл / папка | Назначение |
|--------------|------------|
| `index.html` … `privacy.html` | Страницы сайта |
| `css/style.css` | Стили, адаптив |
| `js/main.js` | Меню, cookie, поиск, отправка формы |
| `server.js` | Express: статика, API заявок и настроек |
| `admin/index.html` | Админ-панель (заявки, настройки) |
| `data/leads.json` | Хранилище заявок |
| `data/settings.json` | Настройки (телефон, email, адрес) |
| `robots.txt`, `sitemap.xml` | SEO |

## Требования по ТЗ

- ✅ Главная (Hero, преимущества, услуги, кейсы, цифры, CTA, контакты)
- ✅ О компании, Услуги/Решения, Отрасль/Фарма, Технологии, Контакты
- ✅ Формы обратной связи, cookie-уведомление, политика конфиденциальности
- ✅ Поиск по сайту, адаптивная вёрстка (320px+), семантика HTML5, базовый SEO
- ✅ Админ: заявки, настройки сайта
- Для продакшена: подключить HTTPS, при необходимости — авторизация в админке, интеграция с CRM/email, аналитика (Яндекс.Метрика, Google Analytics).
